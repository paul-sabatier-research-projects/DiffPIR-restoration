This set of MATLAB files contain an implementation of the algorithms
described in the following paper:

[1] Fast Single Image Super-resolution. Ningning Zhao, Qi Wei, Adrian Basarab, Denis Kouame and Jean-Yves Toureneret.[On line]http://arxiv.org/abs/1510.00143

[2] Qi Wei, Nicolas Dobigeon and Jean-Yves Tourneret, "Fast Fusion of Multi-Band Images Based on Solving a Sylvester Equation," IEEE Trans. Image Process., vol. 24, no. 11, pp. 4109-4121, Nov. 2015.

The code is available at http://zhao.perso.enseeiht.fr/
